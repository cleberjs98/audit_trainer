# Roadmap and V1 Limitations

This document separates implemented V1 behavior from future V2 work.

## Implemented V1

- Supabase Auth login.
- Controlled invitation acceptance.
- Team Management V1 invites, pending invitations, and Cancel invite.
- Optional Resend invitation emails with manual fallback.
- Role-specific Dashboard Analytics V1.
- Start Audit.
- Pret CE V1 guided checklist.
- Review & Complete.
- Audit History V2.
- Manual Action Plans V1.
- Leader own-store action plan management.
- Store Management V2.
- Premium mobile Graphite + Signal Crimson experience.

## Intentional V1 Limitations

- Request access is not implemented.
- Multi-store manager support is not implemented.
- Active user role/scope editing UI is not fully implemented.
- Store manager assignment requires the target profile to already have `profiles.store_id` matching the store.
- Store performance reports use only currently available data.
- Photo upload is not implemented in the current V1 UI.
- AI-generated action plans are not implemented.
- AI dashboard trends/rankings are not implemented.
- PDF export is not implemented.
- Email sending depends on optional Resend configuration; manual invite fallback remains valid.
- Employee/job title/person/team analysis is not implemented.

## V2 Backlog

- Request access workflow.
- Multi-store manager assignments.
- Active user management and reassignment UI.
- Richer store performance reports and trends.
- Employee/team analysis.
- Photo evidence upload and review.
- AI summaries, recommendations, dashboard trends, and rankings.
- PDF export.
- Password recovery UI.
- Area management UI.
- Store bulk import.
